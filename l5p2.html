Image Slide Show

Lab Exercise 5, Part 2






One practical use for a slide show is viewing different angles of an object. We see this all the time when looking at things such as cars, clothing, and furniture pieces. It allows the user to see different angles of the object.
For example, let's say I want to buy a shirt. To ensure that I like the shirt I want to be able to see the front and the back of the shirt. Having a slide show go between the two images would be very useful. If the back of the shirt
had a strange design on it, but I couldn't flip between the images to see it, I would be very disappointed when I recieved the shirt. This also applies to buying a car or viewing pieces of furniture. I want to know that I like every
angle of the object before purchasing.
Another use for a slide show is for companies or organizations that hold events. Placing a slide show onto their site would allow users to cycle through "fliers" and see each event is occuring. Being able to cycle through endlessly
makes sure that the user can go back to the slide of an event if they skip it. I've seen this on websites for libraries, high schools, and colleges. It could also be useful for businesses that do events for their employees.
